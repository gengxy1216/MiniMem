# 小红书文案：OpenClaw + MiniMem 记忆增强

## 标题候选 1
给 OpenClaw 加上“长期记忆”：本地跑、低资源、多角色共享，一键就能用

## 标题候选 2
Agent 不再“健忘”：MiniMem 三层召回 + 实体关系记忆 + 跨会话复用

## 正文（可直接发）
最近把 OpenClaw 接了 MiniMem，体验很像给 Agent 装了一个“可成长的大脑”。

核心感受就 6 点：

1. 资源占用很小  
本地优先架构，轻量部署，边缘设备也能跑。

2. 多角色记忆可控  
支持分角色记忆（`per_role`）和共享记忆组（`shared`）两种模式。  
你可以让 bot A/B 各自独立记忆，也可以让它们共享团队记忆池。

3. 三层记忆召回  
文本 + 向量 + 图谱混合召回，不是只靠关键词命中。  
查“语义相近”内容也能找回上下文。

4. 能记住实体关系  
对“谁和谁有什么关系、谁做了什么”这类信息更友好。  
复杂对话不会只剩碎片。

5. 支持 foresight 时间信息  
时间边界相关的记忆可以进入检索链路，做长期任务更稳。

6. 一键安装 + 自动压缩上下文  
插件支持一键安装，默认就有自动注入和自动写回。  
还能把长会话压缩后写入记忆，减少上下文长度压力。

实际落地上，最喜欢的是它“默认简单、进阶可控”：

- 新手：一条命令安装即可跑
- 进阶：按角色策略、检索策略、自动写回策略精细调

如果你在做多 Agent 协作，或者希望跨会话保持长期记忆，这套组合真的值得试。
