# Security Policy

## Supported Versions

MiniMem is under active development. Please use the latest `main` branch when possible.

## Reporting a Vulnerability

If you find a security issue:

1. Do not open a public issue with exploit details.
2. Use GitHub Security Advisories (preferred) to report privately.
3. If advisories are unavailable, open a minimal issue without sensitive details and request a private channel.

Please include:

- Affected endpoint/module
- Impact and attack scenario
- Reproduction steps
- Suggested mitigation (if any)
